Site web de catalogue de films

Contexte :

Pour mon PPE de première année de BTS SIO, j'ai eu pour mission de créer un site statique qui met en avant une collection de films en utilisant HTML/CSS, PHP et MySQL.

Objectif :

L'objectif principal de ce projet est de permettre aux utilisateurs d'accéder à un catalogue de films.

Description fonctionnelle :

Les principales fonctionnalités du site incluent la possibilité de visualiser les informations détaillées d'un film, de filtrer les recherches pour trouver des titres spécifiques, de visionner la bande-annonce d'un film, de modifier les données d'un film existant et d'ajouter de nouveaux films à la collection.

Ressources et Contraintes :

Pour la réalisation de ce projet, j'utilise mes connaissances en développement web, Visual Studio Code comme environnement de développement, et je stocke les données dans une base de données MySQL. Les technologies utilisées incluent HTML, CSS, PHP, et MySQL pour atteindre les objectifs du projet.
