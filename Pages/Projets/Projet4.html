Projet de Stage 2ème année

Contexte :

Pendant ce stage j'ai intégré une équipe de développeurs travaillant sur une application de dashboard pouvant afficher plusieurs widgets, comme l'état des environnements d'applications, des newsletters, etc.

Objectif :

L'objectif principal de mon stage était d'intégrer des widgets dans un tableau de bord web. Ces widgets devaient afficher le dernier message d'un canal Teams dans un cas, et un iframe contenant un post LinkedIn dans l'autre.

Description fonctionnelle :

Le projet vise à développer une application de tableau de bord permettant aux utilisateurs de visualiser des données, telles que l'état des environnements d'applications et des newsletters. Les fonctionnalités principales comprennent la visualisation des détails des widgets, la recherche et le filtrage des informations, la modification et l'ajout de nouveaux widgets.

Ressources :

Pour la réalisation de ce projet, les ressources utilisées comprennent Spring, ainsi que l'utilisation de l'IDE IntelliJ pour le développement backend. Du côté du frontend web, les technologies Angular sont employées, avec Visual Studio Code comme environnement de développement. La base de données MongoDB est utilisée pour le stockage des données.
